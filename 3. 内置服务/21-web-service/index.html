Web Service(Web服务)

webservice已经被转移到一个单独的项目中。

地址是:

https://github.com/scrapy-plugins/scrapy-jsonrpc

下面是我自己翻译带过来的

scrapy-jsonrpc

Scrapy -jsonrpc是通过JSON-RPC控制一个正在运行的Scrapy web爬虫程序的扩展。该服务通过JSON-RPC 2.0协议提供对主Crawler对象的访问。

安装

使用pip安装scrapy-jsonrpc:

$ pip install scrapy-jsonrpc

配置

首先，您需要在settings.py中打开对EXTENSIONSdict的注释。例如:

EXTENSIONS = {
    'scrapy_jsonrpc.webservice.WebService': 500,
}

然后，需要使用JSONRPC_ENABLED设置启用extension，设置为True。

web服务器将侦听JSONRPC_PORT中指定的端口(默认情况下，它将尝试侦听端口6080)，并将登log到JSONRPC_LOGFILE中指定的文件。

访问crawler对象的端点为:

http://localhost:6080/crawler

客户端实例

为了说明如何构建client，提供了一个命令行工具。您可以在example-client.py中找到它。它支持一些基本命令，如列出运行中的spider等。

Settings

以下是控制web服务行为的设置:

JSONRPC_ENABLED

默认: False

一个布尔值，用于指定是否启用web服务(如果它的扩展也是启用的)。

JSONRPC_LOGFILE

默认: None

用于记录对web服务发出的HTTP请求的文件。如果未设置web，日志将发送到标准的scrapy日志。

JSONRPC_PORT

默认: [6080, 7030]

用于web服务的端口范围。如果设置为None或0，则使用动态分配的端口。

JSONRPC_HOST

默认: '127.0.0.1'

web服务应该监听的接口。
